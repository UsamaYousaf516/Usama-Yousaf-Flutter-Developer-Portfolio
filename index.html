Elite Fitness App — Gym Member Experience

A membership-only fitness app for a gym client: members build and manage workouts from a large exercise library, track history, visualize progress with charts, and use a full nutrition module for daily intake, recipes, and meal ideas matched to their preferences.

Problem solved: The gym needed one branded product that tied paid membership to training and nutrition — replacing ad-hoc spreadsheets and generic apps with access control, structured workouts, and credible nutrition data in a single Flutter experience.
  • Workout CRUD with a vast exercise catalog (ExerciseDB via RapidAPI)
  • Workout history, analytics, and charts for long-term progress
  • Nutrition tracking with FatSecret API — intake, recipes, and meal inspiration
  • Member eligibility and gym user flows via Gymmater and Gatekeeper APIs
  • Supabase-backed data layer for a scalable, real-time friendly backend

Restart Fitness — Level-Up Training Platform

A fitness product built around progression: ten levels, each with assigned 30-day workouts and meal plans, level tests with video submissions, and admin review — plus weekly challenges where members submit proof and top performers are crowned.

Problem solved: Static programs do not scale accountability. This system gamifies adherence — clear levels, admin-verified tests, and community challenges — while a React web admin panel orchestrates assignments, reviews, and winners.
  • 10-level journey: new users start at level 1 with structured plans
  • Admin web app (React) assigns workouts, meals, tests, and reviews submissions
  • Video-based level tests; admins approve or reject to unlock the next level
  • Weekly challenges (e.g. 100 pushups, 100 squats) with video proof
  • Admin selects top 3 challenge winners from submissions
